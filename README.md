# 🎡 김길현의 두비두 룰렛 (Canvas Roulette Tool)

> HTML Canvas를 활용하여 스트리머 김길현의 두비두(식재료) 룰렛 콘텐츠를 매끄럽게 진행하기 위해 구현한 단일 파일 기반의 프론트엔드 웹 애플리케이션입니다.

브라우저에서 즉시 실행 가능한 독립형(Standalone) 스크립트로 개발되어 누구나 간편하게 사용할 수 있습니다.

---

### 🚀 주요 기능 (Key Features)

- **안정적인 룰렛 기능:** 많은 양의 항목도 렉 없이 부드럽게 회전합니다.
- **당첨 슬롯 관리:** 슬롯 추가 및 재추첨(Undo/되돌리기) 로직이 완벽하게 지원됩니다.
- **데이터 보관:** 룰렛에 추가된 요소명과 개수를 텍스트로 편리하게 보관할 수 있습니다.
- **부가 기능:** 진행을 돕는 간단한 메모 패널 및 스톱워치 기능이 내장되어 있습니다.

---

### 🛠️ 시작하기 (Getting Started)

본 프로그램은 가벼운 웹 소스로 구성되어 있어 별도의 설치 과정이 필요하지 않습니다.

👉 **[최신 버전 다운로드 받기](https://github.com/halgurx/doobidoo/archive/refs/tags/v1.0.0.zip)**

1. 위 링크를 클릭해 압축 파일을 다운로드합니다.
2. 압축을 해제한 뒤 `rullet.html` 파일을 더블 클릭하여 브라우저에서 실행합니다.
3. ⚠️ **주의:** 모든 파일과 폴더가 동일한 위치에 유지되어야 정상적으로 동작합니다.

### 📦 파일 구조 (Project Structure)

```text
├── rullet.html          # 메인 웹 페이지 구조 및 레이아웃
├── css/
│   └── rullet.css       # 애플리케이션 스타일 및 UI 레이아웃
├── js/
│   ├── rullet.js        # 메인 로직 및 Canvas 제어 스크립트
│   └── confetti.js      # 시각 효과를 위한 로컬 라이브러리 파일
└── sound/               # 효과음 폴더 (사용자 커스텀 가능)
```

---

### 🎨 커스텀 방법 (Customization)

#### 1. 효과음 추가 (Sound Customization)
프로젝트 내의 `sound/` 폴더 내부에 커스텀 사운드 파일(`.mp3`)을 채워주세요. 효과음 파일이 없어도 핵심 기능은 동작하지만, 추가하시면 훨씬 더 생동감 넘치는 연출이 가능합니다.
- **파일명:** `1.mp3` ~ `12.mp3` 형식으로 이름을 지정합니다.
- **권장 길이:** 재생 시간은 1~3초 내외가 가장 적당합니다.

> **💡 더 많은 효과음을 사용하고 싶다면?** > `/js/rullet.js` 파일에서 `playWinSound` 함수를 검색한 뒤 아래 코드를 찾아보세요.  
> ```javascript
> const randomNum = Math.floor(Math.random() * 12) + 1;
> ```  
> 위의 숫자 `12`를 원하는 효과음 총 개수에 맞게 변경하고 파일을 추가하시면 됩니다.

#### 2. 스타일 및 로직 변경 (CSS, JS)
- **CSS:** 색상 테마는 2가지로 구분되어 있습니다. 테마 그룹 내의 CSS 컬러 변수를 수정하여 원하시는 색상으로 쉽게 변경할 수 있습니다. 
- **JS:** 자유로운 수정 및 기능 확장이 가능합니다. 단, 파일 상단의 **라이선스 주석만 수정하지 않고 그대로 유지**해 주세요.
- *스스로 코드를 수정하기 어렵다면 AI 툴의 도움을 받거나 개발 지식이 있는 주변 지인에게 요청하시는 것을 추천합니다.*
